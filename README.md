# 📋 Registration Form

This project is a **simple Registration Form** built using **HTML, CSS, and JavaScript**.  
The form collects user details and performs **real-time validation** to ensure that the input fields are filled correctly before allowing submission.

---

## 🚀 Features

- User-friendly **registration form UI**
- **Real-time validation** using JavaScript
- Displays **error messages** when fields are incorrect
- **Submit button remains disabled** until all inputs are valid
- Clean and simple **CSS styling**

---

## 🧾 Form Fields

The registration form includes the following fields:

- **Name** – cannot be empty  
- **Email** – must contain a valid email format  
- **Password** – must contain at least **6 characters**

---

## ⚙️ Technologies Used

- **HTML** – Structure of the form  
- **CSS** – Styling and layout  
- **JavaScript** – Form validation and interactivity
